# Extension Trait

Extension Trait is probably the most amazing feature I've ever seen in a programming language. Simply put, it allows the introduction of methods to any data type. Yes, any data type. Extension traits works by using `traits`. These traits can be defined for external and internal data types. Once the trait is imported that type gains access to the methods defined in there.
